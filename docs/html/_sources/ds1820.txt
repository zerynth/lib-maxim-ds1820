******
DS1820
******

This module implements the class DS1820 to read OneWire temperature sensors DS18b20 and DS18s20
connected to a DS2482 I2C-OneWire bridge.

    
============
DS1820 class
=============

.. class:: DS1820(serial,ow)
        
        Creates a DS1820 instance using the OneWire bridge *ow* with OneWire serial id *serial*.

    
.. method:: read()

        Return the temperature readings as float applying the correct conversion for different DS1820 models.
        If the serial is wrong, raises *UnsupportedError*.
